# Demographic Data Analyzer

## Overview

This project was completed as part of the FreeCodeCamp Data Analysis with Python certification.

The objective is to analyze demographic data from the 1994 U.S. Census database using Python and Pandas and answer several statistical questions about education, income, occupation, race, and working hours.

## Technologies Used

* Python 3
* Pandas

## Dataset

The project uses the Adult Census Income dataset from the UCI Machine Learning Repository.

## Features

The analysis includes:

* Race distribution
* Average age of men
* Percentage of people with a Bachelor's degree
* Income analysis based on education level
* Minimum working hours analysis
* Country with the highest percentage of high-income individuals
* Most popular occupation among high-income individuals in India
